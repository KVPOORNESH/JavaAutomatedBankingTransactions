Problem Statement
Manual banking transactions are often slow and result in errors that affect user convenience and security. There is a need for a fast, reliable, and easy-to-use system to automate routine banking operations using Java.

Scope of the Project
Build a console-based application that allows users to create accounts, deposit, withdraw, transfer money, and view account details.

Ensure quick response times and safe, synchronized transactions.

Focus on accuracy, reliability, and user-friendly operation.

Designed for student learning and demonstration of Java concepts.

Target Users
Students learning Java programming and object-oriented design.

Anyone interested in understanding basic banking systems.

Users who wish to perform banking operations with speed and simplicity.

High-Level Features
Account creation and management

Deposit, withdraw, transfer operations

Error handling for failed transactions

Viewing account details and balances

Simple, menu-based interaction

Modular code (Account, Transaction, Bank classes)

Ready for extension into more advanced banking solutions
